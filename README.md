# FlowDesk — AI-Generated Landing Page

A responsive one-page marketing website for **FlowDesk**, a fictional productivity app that helps remote teams plan their week.

## Project Structure

```
ai-generated-landing-page/
├── index.html
├── style.css
└── README.md
```

## Sections

| Section  | Description |
|----------|-------------|
| Header   | Sticky navigation bar with logo and links |
| Hero     | Headline, subtext, CTA buttons, and a decorative UI mockup card |
| Features | 4-column grid of feature cards (stacks on mobile) |
| CTA      | Call-to-action section with sign-up prompt |
| Footer   | Brand tagline, navigation links, copyright |

## Technologies Used

- Semantic HTML5 (`header`, `nav`, `section`, `article`, `footer`)
- CSS Flexbox (navigation, hero layout, CTA, footer)
- CSS Grid (feature cards — 4 columns → 2 → 1 on mobile)
- CSS Media Queries for responsive design (`@media max-width: 768px` and `480px`)
- Google Fonts — Inter

## AI Assistance Note

AI was used to:
- Suggest the app name **FlowDesk** and its one-line value proposition ("Plan Your Week, Own Your Time")
- Draft the hero headline, subtext, and feature card copy
- Suggest the colour palette (Indigo `#4f46e5` primary, neutral greys)
- Recommend the Inter font pairing and overall layout structure

All HTML and CSS was written from scratch following the 3MTT NextGen Cohort project brief.

## How to Run

1. Open the project folder in VS Code.
2. Right-click `index.html` → **Open with Live Server** (or open directly in a browser).
3. Resize the browser window or use DevTools device toolbar (F12) to test mobile responsiveness.
